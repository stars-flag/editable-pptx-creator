# HTML Parsing Guide

This guide explains how to parse HTML presentations for conversion to editable PPTX.

## Overview

The HTML parser extracts structured data from HTML presentations generated by slide-creator or compatible tools. It identifies slide sections, content types, and styling information to build native PowerPoint objects.

## Supported HTML Structure

### Basic Slide Structure

```html
<section class="slide">
  <!-- Slide content -->
</section>
```

Each `<section class="slide">` represents one PowerPoint slide.

### Content Types

#### 1. Title Slide

```html
<section class="slide">
  <h1>Main Title</h1>
  <p class="subtitle">Subtitle text</p>
</section>
```

**Extracted data:**
```python
{
  "type": "title",
  "title": "Main Title",
  "subtitle": "Subtitle text"
}
```

## Parsing Rules

### 1. Slide Detection

```python
slides = soup.find_all('section', class_='slide')
```

### 2. Title Detection

```python
# H1 = main title (title slide)
h1 = slide.find('h1')
if h1:
    title = h1.get_text(strip=True)
    title_level = 1

# H2-H6 = section titles (content slides)
h2 = slide.find('h2')
if h2:
    title = h2.get_text(strip=True)
    title_level = 2
```

### 3. Content Type Detection

```python
# Check for subtitle (title slide)
subtitle = slide.find(class_='subtitle')

# Check for list items (content slide)
list_items = slide.find_all('li', class_='list-item')

# Check for cards (card grid slide)
cards = slide.find_all('div', class_='card')

# Check for color cards (color palette slide)
color_cards = slide.find_all('div', class_='color-card')

# Check for poems (poem slide)
poems = slide.find_all('div', class_='poem')

# Check for images (image slide)
image = slide.find('img', class_='slide-image')
```

### 4. Color Extraction

```python
# Extract from inline styles
style = element.get('style', '')
if 'background' in style or 'color' in style:
    # Parse color value
    color = extract_color_from_style(style)

# Extract from CSS variables
css_vars = extract_css_variables(soup)
primary_color = css_vars.get('--color-primary')
```

### 5. Text Cleaning

```python
# Strip whitespace
text = element.get_text(strip=True)

# Remove extra spaces
text = ' '.join(text.split())

# Preserve line breaks for poems
if element.get('class') == ['poem-text']:
    text = element.get_text('\n', strip=True)
```

## CSS Variable Extraction

```python
def extract_css_variables(soup):
    """Extract CSS custom properties from style tags"""
    variables = {}

    style_tags = soup.find_all('style')
    for style_tag in style_tags:
        css_text = style_tag.get_text()

        # Find :root variables
        root_match = re.search(r':root\s*{([^}]+)}', css_text)
        if root_match:
            root_content = root_match.group(1)

            # Extract variable declarations
            var_pattern = r'--([\w-]+)\s*:\s*([^;]+);'
            for match in re.finditer(var_pattern, root_content):
                var_name = match.group(1)
                var_value = match.group(2).strip()
                variables[var_name] = var_value

    return variables
```

## Color Conversion

```python
def hex_to_rgb(hex_color):
    """Convert hex color to RGB tuple"""
    hex_color = hex_color.lstrip('#')
    return tuple(int(hex_color[i:i+2], 16) for i in (0, 2, 4))

def rgb_to_pptx(rgb_tuple):
    """Convert RGB tuple to python-pptx RGBColor"""
    from pptx.dml.color import RGBColor
    return RGBColor(*rgb_tuple)
```

## Error Handling

```python
# Handle missing elements
if not title:
    title = f"Slide {slide_index}"

# Handle empty lists
if not items:
    items = []

# Handle invalid colors
try:
    rgb = hex_to_rgb(color_hex)

```

## Best Practices

1. **Be lenient**: Don't fail on missing elements, use defaults
2. **Preserve hierarchy**: Maintain the structure of the original HTML
3. **Extract styling**: Capture colors, fonts, and spacing from CSS
4. **Handle edge cases**: Empty slides, missing titles, invalid colors
5. **Log warnings**: Inform user about parsing issues without failing

## Integration with PPTX Builder

The parsed data is passed to the PPTX builder:

```python
parser = HTMLParser()
slide_data = parser.parse('presentation.html')

builder = PPTXBuilder(theme)
builder.build(slide_data)
builder.save('presentation.pptx')
```
